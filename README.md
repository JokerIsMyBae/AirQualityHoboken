# AirQualityHoboken
Applied IoT project voor luchtkwaliteitsmetingen in Hobokense Polder


### Doel
Meten van specifieke luchtkwaliteitsparameters:
- CO2
- TVOC
- NOx
- Fijnstof
- Temperatuur
- Luchtvochtigheid
- Luchtdruk
